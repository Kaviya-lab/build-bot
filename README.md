## AI-Based Learning Management System for Electronic Projects

Build-bot is a browser-based learning platform that helps students and hobbyists explore electronics projects using AI-powered image recognition. 
Users can upload an image of electronic components, and the system analyzes it to suggest relevant learning projects and ideas.

## Features

Upload an image of electronic components via the web UI

Analyze the image using a trained Teachable Machine (TensorFlow.js) model

Display detected objects with confidence scores

Suggest relevant electronics project ideas based on detected components

Fully browser-based — no backend required

## Tech Stack

Frontend: React

AI / ML: TensorFlow.js, Google Teachable Machine

Deployment: Runs entirely in the browser

## How It Works

User uploads an image of electronic components

The Teachable Machine model detects objects in the image

Confidence scores are generated for each detected item

The system maps detected components to relevant learning project ideas

Results are displayed instantly in the browser
